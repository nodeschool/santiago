Sé parte de la experiencia

NodeSchool es una iniciativa global llevada a cabo por voluntarios para la comunidad con dos objetivos: crear planes de estudio de programación de alta calidad y organizar eventos de aprendizaje comunitario.

Ven a aprender JavaScript y NodeJS con workshops interactivos, ¡gratis!.

¿Qué necesito para participar?

  • Un equipo con NodeJS v4 o v5 instalado. Descárgalo desde el sitio oficial.
  • Tu editor de texto favorito. Sublime Text, Atom, Vim, Emacs, Visual Studio Code, etc.
  • Una terminal de tu preferencia. Todos los S.O. vienen con una terminal propia.
  • Una cuenta de GitHub (opcional)

Agenda

javascripting

Aprende lo básico de JavaScript. No necesitas experiencia previa programando.
npm install -g javascripting

learnyounode

Aprende lo básico de NodeJS: código asíncrono, i/o y HTTP.
npm install -g learnyounode

stream-adventure

Aprende a componer streaming interfaces con .pipe().
npm install -g stream-adventure

expressworks

Aprende lo básico del framework Express.js.
npm install -g expressworks

Otros workshops disponibles para usuarios más avanzados.
